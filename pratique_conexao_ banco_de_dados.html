Treinamento de Digitação - Conexão com Banco de Dados

Código de Conexão com Banco de Dados em PHP e MySQL

Digite o código abaixo:

<?php
$servername = "localhost";
$username = "seu_usuario";
$password = "sua_senha";
$dbname = "seu_banco_de_dados";

$conn = mysqli_connect($servername, $username, $password, $dbname);

if (!$conn) {
 die("Falha na conexão: " . mysqli_connect_error());
}

echo "Conexão com o banco de dados estabelecida com sucesso!";

mysqli_close($conn);
?>

Escreva o código acima na área abaixo:

  Nesta página, você pode digitar o código de conexão com o banco de dados na área de texto abaixo do código original. À medida que você digita, a precisão do seu código em relação ao original será calculada e exibida acima da área de texto. Espero que isso atenda às suas necessidades!
