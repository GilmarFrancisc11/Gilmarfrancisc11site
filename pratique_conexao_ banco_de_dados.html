Treinamento de Digitação - Conexão com Banco de Dados

Código de Conexão com Banco de Dados em PHP e MySQL

Digite o código abaixo:

<?php
$servername = "localhost";
$username = "seu_usuario";
$password = "sua_senha";
$dbname = "seu_banco_de_dados";

$conn = mysqli_connect($servername, $username, $password, $dbname);

if (!$conn) {
 die("Falha na conexão: " . mysqli_connect_error());
}

echo "Conexão com o banco de dados estabelecida com sucesso!";

mysqli_close($conn);
?>

Pratique digitando o código acima no teclado, tentando não olhar para ele. Isso ajudará a melhorar sua habilidade de digitação e familiaridade com o código de conexão com banco de dados em PHP e MySQL.
