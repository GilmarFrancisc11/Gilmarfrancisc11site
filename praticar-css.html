  Nesta versão atualizada, removi a classe  .highlight-text  do segundo campo ( css-selector ) e adicionei o atributo  readonly  para torná-lo somente leitura. Agora, quando você digitar no primeiro campo ( css-properties ), a cor verde será aplicada no segundo campo ( css-selector ) à medida que você digita. As palavras correspondentes serão destacadas em verde. Espero que isso atenda às suas necessidades! Se você tiver mais dúvidas, é só me perguntar.
