  Neste exemplo, adicionei um segundo campo de entrada    onde você pode digitar um seletor CSS. Quando você clica em "Aplicar", o seletor é usado para selecionar os elementos correspondentes no primeiro campo de propriedades CSS. Os elementos selecionados recebem a classe CSS "selected", que tem um estilo de fundo azul claro. Você pode experimentar diferentes seletores CSS para ver os elementos sendo selecionados. Certifique-se de digitar corretamente o seletor CSS para obter os resultados desejados. Espero que isso seja útil para você praticar CSS! Se você tiver mais dúvidas, é só me perguntar.
